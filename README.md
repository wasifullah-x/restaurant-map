# Restaurant Map

A React + Vite app to explore halal-friendly restaurants on an interactive map, with search and filtering.

## Run Locally

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Open the local URL shown in your terminal (usually http://localhost:5173).

## Build

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
